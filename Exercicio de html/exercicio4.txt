<h1>ID e Class</h1>
<p id="sel1">O id serve como um identificador único,
ou seja em todo o seu HTML só pode ser usado esse id uma vez, 
ele é mais usado para menus, e em javascript.
Para ser criado um id no arquivo css deve ser colocado um # na frente do nome.
Já o class pode ser usado quantas vezes se quizer,
o que permite se atribuir um layout a muitas partes.
Para ser criado uma class no arquivo css deve ser colocado um . (ponto) na frente do nome.
Sendo então usados para identificar elementos. </p>
<br>
<h2>Div e SPAN</h2>
<p id="sel2">A diferença entre span e div é que span é um elemento inline e geralmente é usado para um pequeno bloco de HTML,
e div é um elemento de bloco (o que basicamente é equivalente a ter uma quebra de linha antes e depois do elemento)
e é usado para agrupar blocos maiores de código (geralmente divisões da página como menu cabeçalho, etc.)
Sendo usados para agrupar elementos de um mesmo grupo.</p> 
<br>
<h3>CSS interno e Externo</h3>
<p id="sel1">O CSS Externo e usado para criar folhas em diferentes estilos,fora da pagina em que sera utilizado,
podendo ssim ser utilizado em mas de uma pgina.
Já o CSS interno e utilizado dentro da pagina em que sera utilizado,podemdoassim ser utilizado em uma unica pagina.
Assim sendo o melhor para ser utilizado e o Externo,pois atravês de uma unica pagina pode se modificar e ver seus erros.
No interno e mas rapido a sua confecção de pagina.Mas ambos tem o poupoamento de tempo,ganho de flexibilidade,paginas leves de facil visualização.
Rapides na coreção de erros,aumento da consistência,Controle de Leyout e grafico.</p>
